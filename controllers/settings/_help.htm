Configuration Instructions

  1. Download the Eeze app for Android or iOS
  2. Go to https://eeze.io/login and scan the code with the app to register for an account
  3. Create a new Domain for your site, using as the value for the "Website" field.
    • Note: Make sure you have HTTPS setup and are using it first.
  4. Copy the Client ID and Client Secret from the success page and paste them in these fields.
    • Note: These values will not be visible after navigating away from that page so make sure you copy them down first!
  5. Choose a role that you want new users to be enrolled with by default
  6. Click the Save button
  7. That's it! Logging in with Eeze has been fully configured!
